◆ localizedKeys()

Keyboard::Key sf::priv::HIDInputManager::localizedKeys ( UniChar  ch)
static

Try to convert a character into a SFML key code.

Return sf::Keyboard::Unknown if it doesn't match any 'localized' keys.

By 'localized' I mean keys that depend on the keyboard layout and might not be the same as the US keycode in some country (e.g. the keys 'Y' and 'Z' are switched on QWERTZ keyboard and US keyboard layouts.)
