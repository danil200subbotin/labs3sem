Detailed Description

Класс-контейнер, отавечающий за работу всего вместе.

Содержит: Размеры Карты, Указатель на первую дорогу от логова, Указатель на замок, Указатель на логово

Содержит списки: Простых башен, Магических башен, ловушек, леса, дорог, врагов

Конструктор данного класса создает необходимые для игры вещи
